# Система учёта пациентов (Patient Management System)

Приложение для ведения учета пациентов с возможностью визуализации статистических данных. Проект реализован на Python с использованием библиотеки `tkinter` для графического интерфейса и `matplotlib` для построения графиков.

## Функционал

*   **Управление данными:**
    *   Добавление новых пациентов.
    *   Редактирование существующей информации.
    *   Удаление записей.
    *   Автоматический расчет ИМТ (Индекса Массы Тела).
    *   Генерация тестовых данных (используется библиотека `Faker`).
*   **Хранение данных:**
    *   Все данные автоматически сохраняются в файл `patients.json` и загружаются при запуске.
*   **Статистика и визуализация:**
    *   Круговая диаграмма распределения по полу.
    *   Гистограмма распределения по возрасту.
    *   Boxplot (ящик с усами) для ИМТ с учетом пола.
    *   Scatter plot (точечная диаграмма) зависимости ИМТ от возраста.

## Технологии

*   **Python**
*   **Tkinter** — графический интерфейс.
*   **Matplotlib** — построение графиков.
*   **Pillow (PIL)** — работа с изображениями (отрисовка иконок).
*   **Faker** — генерация случайных данных (опционально).

## Как пользоваться

1.  **Главное окно:** Отображает таблицу со списком всех пациентов (ФИО, Возраст, Пол, Рост, Вес, ИМТ).
2.  **Кнопки управления (слева):**
    *   **Добавить:** Открывает форму для ввода данных нового пациента.
    *   **Изменить:** Позволяет редактировать данные выделенного в таблице пациента.
    *   **Удалить:** Удаляет выбранного пациента.
    *   **Статистика:** Открывает отдельное окно с четырьмя графиками аналитики.
    *   **Генерация:** Создает 5 случайных пациентов для теста.
